# sim
Java Trading Simulator  
Tested on openjdk 17.0.6 2023-01-17 LTS  

## Install and Run
git clone https://github.com/cr35wx/sim/  
cd sim/src/  
javac Main.java  
java -classpath . Main  
  
OR  
Clone the project, open it in Intellij, configure a JDK, and run main.
